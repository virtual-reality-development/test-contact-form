# Test - Contact Form

### Requirements

```
路 HTML5
路 CSS3 / Responsive / Bootstrap
路 Built-in form validation (HTML5) / Client-Side validation
路 Vanilla JavaScript or with Vue.js
路 Upload the result to your repository
路 Time: max. 3 days
```

### The contact form should have at least the following fields
``` 
路 Name (Required)
路 Email (Required) + custom regex validation
路 Phone (Optional)
路 Message (Required)
路 Confirm Policies (Required)
路 Button to send form
```

### Behavior
> When sending the form and all the fields are correctly validated, an alert window should appear with the message:  "__Successfully sent!__"  and in the console a JSON object with all the form data.
