# Test - Contact Form

### Requirements

``` 
路 HTML5
路 CSS3 / Responsive
路 VueJs and/or with Vanilla JavaScript
路 Client-Side validation
路 Upload the result to your repository
``` 

### The contact form should have at least the following fields
``` 
路 Name (Required)
路 Email (Required / Regex Validation)
路 Phone (Optional)
路 Message (Required)
路 Confirm privacy policy (Required)
路 Send form button
```

### Behavior
> When sending the form and all the fields are correctly validated, an alert window should appear with the message:  "__Successfully sent!__"  and in the console a JSON object with all the form data.
